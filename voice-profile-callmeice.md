## SECTION 1: BELIEFS & CONTRARIAN TAKES

### Q1: มีความเชื่ออะไรเกี่ยวกับงานหรือวงการของคุณ ที่ถ้าพูดออกไปดังๆ ในวงข้าวกับคนในวงการเดียวกัน คนจะเงียบกริบ?

จากสิ่งที่ผมจะพูดเนี่ยก็คือ Prompt อ่ะไม่ได้ดีเท่า Context ที่ชัด ซึ่งแต่ก่อนน่ะ มันไม่ค่อยมี แต่เดี๋ยวเนี้ยมันเริ่มดังขึ้นเรื่อยๆแล้ว ผมก็เลยไม่รู้ว่าผมจะพูดอะไรดี เพราะว่าสิ่งที่ผมเชื่อก็คือ Prompt มันไม่ได้ดีกว่า Context ที่ชัด

[ขยายเลเยอร์ลึก:] คนพูดว่า context สำคัญ แต่งานที่สำคัญจริงๆ คือ context ที่ดีสำหรับผมคือ Context อันนี้อันเดียวเลยก็คือ About Me มันจะบอกว่าคุณคือใคร ทำอะไร เขียนอะไร อะไรต่างๆที่คุณกำลังสัมภาษณ์ผมอยู่ และผมจะบอกก็คือนี่มันไม่ใช่ไฟล์แรกที่ผมให้คุณสัมภาษณ์ แต่อันนี้ผมมาทำใหม่เพราะว่าความคิดและมุมมองของผมเนี่ยมันไม่เหมือนเดิม

### Q2: Hot Take ที่คุณยอมเถียงจนตัวตาย

Context ดีกว่า Prompt — Prompt Engineering is dead. Context is king.

นั่นแหละ ผมยอมเถียงเลยว่าคุณจะ Prompt ดีตายห่าตายโหงยังไง มันก็ไม่มีทางดีกว่า Context ที่ชัดหรือ Prompt สั้นๆ คือจะต่อให้ prompt ยาวขนาดไหน ยาวเป็นกิโลเมตร แต่มันไม่มี Context มันไม่มีเกี่ยวกับตัวคุณ ไม่มีอะไรต่างๆที่เป็นตัวคุณจริงๆ แบบนั้น AI มันจะทำงานมาให้ตรงกับคุณยังไงล่ะ

### Q3: ภูมิปัญญาที่คนเชื่อกันทั่วไป ที่คุณคิดว่ามันผิด

ผมคิดว่ามันผิด คือโตขึ้นไปตั้งใจเรียน, ไปเป็นเจ้าคนนายคน, ไปเป็นข้าราชการ, ทำงานเงินเดือนสูงๆ ทั้งที่จริง อ่ะ มันเป็นคำพูดเกาะหูที่ผมรู้สึกว่ามันไม่ได้ดีอะไรเลย

อีกอย่าง มันควรที่จะเป็นเรื่องของเด็กด้วยว่า โอเค โตขึ้นมา เขาอยากจะเป็นอะไรกันแน่ ถ้าเขาไม่อยากจะเป็นเจ้าคนนายคนอย่างที่คนอื่นบอก, อย่างที่แม่บอก, พ่อแม่บอกนู่นนี่นั่น ไปเป็นข้าราชการอะไรเนี่ย แต่ถ้าเด็กอยากจะเป็นนักธุรกิจขึ้นมาล่ะ จริงไหม แต่พ่อแม่มาขัดขวางอย่างเงี้ย หรือว่าถูกคำสั่งสอนอย่างเงี้ย พูดมาเรื่อยๆ คุณคิดว่าเด็กมันอยากจะเป็นไหมล่ะ

นั่นแหละสิ่งที่ผมอยากจะคัดค้านเลย คือผมไม่อยากจะให้คำแบบเนี้ยมาใช้กับคนรุ่นใหม่เลย ควรที่จะมีสิทธิเสรีภาพในตัวเองอยู่แล้ว

### Q4: โมเมนต์ที่คุณโดนแบบนี้กับตัวเอง

ก็ว่าเพราะว่าผมก็เคยโดนนั่นแหละ แต่ว่าเดี๋ยวเนี้ยผมก็ไม่ค่อยโดนแล้ว เพราะว่าผมยืนหยัดที่จะทำเรื่องเรื่องนี้ก็คือ ทำธุรกิจ ก็ไม่มีใครพูดหรอก แม่ผมก็บอกว่าโตขึ้นไปเป็นตำรวจเป็นนายร้อยนะ แต่ว่าผมไม่เอา ยายก็บอกเออไปบวชไหมไปนู่นไปนี่ แต่ผมไม่เอา แล้วก็พ่อผมก็บอกว่า เออมาช่วยงานแกหน่อย ผมยิ่งไม่ชอบ

ก็คือไม่รู้ดิ ผมก็จำไม่ได้แล้วเหมือนกัน ทำไมผมถึงเป็นอย่างนี้ แต่อาจจะเป็นเพราะว่าสภาพแวดล้อมแล้วก็สิ่งต่างๆ ที่อยู่รอบตัวมันเลยทำให้รู้สึกว่าผมไม่ชอบ

### Q5: "สูตรสำเร็จ" ในวงการธุรกิจ/ผู้ประกอบการที่คุณคิดว่าเป็น bullshit

ผมคิดว่า Bullshit สุดก็น่าจะประมาณพวกตื่นให้เร็ว ตื่นให้เช้าๆ อะไรประมาณนั้น ซึ่งจะตื่นเช้ามาทำไมถ้าคุณไม่มีลูกค้าหรือว่าไม่มีอะไรเลย ทั้งที่คุณสามารถตื่นหกโมงเจ็ดโมงแล้วก็ทำอะไรเพื่อธุรกิจก็ได้ แทนที่จะตื่นมาตีสี่แล้วก็มานั่งทำงาน ผมคิดว่ามันน่าจะ bullshit เกินไป

ผมยกตัวอย่างง่ายๆ สั้นๆ เลย ถ้าคุณมีงาน คุณตื่นตีสี่ อันเนี้ยมันจะพอเข้าใจอยู่ แต่ถ้าคุณทำงานอย่างเดียว คุณไม่สนใจร่างกายสุขภาพร่างกายตัวเอง แบบเนี้ยมันจะไปได้ที่ไหนล่ะ

### Q6: ครีเอเตอร์/คอร์ส/กูรูที่คุณเคย buy in แต่ตอนนี้รู้สึกว่ามั่ว

ไม่มีครับ แต่ว่ามันจะมีคนคนนึงที่สอนผมเกี่ยวกับเรื่อง Mindset แล้วผมก็เชื่อ แต่ว่ามันเป็น Mindset ที่ดู Ego เกินไปจนทำให้รู้สึกว่าแบบเออ ดูไม่ค่อยใช้ได้เลยแฮะ ก็มันเกี่ยวกับพวก mindset ที่ดี การมี mindset ที่ดี การมี mindset ที่แข็งแกร่ง อะไรประมาณนั้น

### Q7: ระบบโรงเรียนไทย vs สิ่งที่คุณเรียนรู้จากการทำธุรกิจจริงๆ

ผมคิดว่าสิ่งที่ระบบสอนกับสิ่งที่ผมได้เรียนรู้มามันต่างกันมากเลย

อย่างแรกคือเรื่อง Mindset คือโรงเรียนเนี่ยมันจำกัดกรอบ กรอบในเรื่องไอเดีย, ความคิดของเด็ก, สิ่งที่เด็กอยากจะเป็น แต่เขาจำกัดกรอบของนักเรียนวัดผลด้วยที่คะแนน ว่าแบบเออคะแนนคุณต้องดีนะ คุณต้องมีชีวิตที่ดี เอ้ย มันจะดีแน่นอนถ้าคุณตั้งใจเรียน เรียนเก่งๆ สอบได้คะแนนดีๆ แต่ผมคิดว่าไม่ ผมคิดต่างออกไป

เพราะว่าอย่างแรกเลยก็คือ โรงเรียนสอนมาให้ตั้งใจเรียน ให้อยู่ภายใต้การควบคุมของนักของโรงเรียน แล้วนักเรียนของเขาเนี่ยจะมีความคิดที่ว่า "เออ คุณน่ะต้องเรียนให้เก่งๆ ถึงจะประสบความสำเร็จในชีวิตนะ" แต่ที่จริงไม่ใช่เลย ความประสบความสำเร็จในชีวิตเนี่ยมันไม่ได้วัดผลที่คะแนนใบเกรด แต่มันวัดว่าใครมันตั้งเป้าหมายประสบความสำเร็จ ต่างออกไป มันไม่เหมือนกันเลยประสบความสำเร็จในแต่ละของ ของคนอื่นๆ

อย่างเช่น ผม ผมประสบความสำเร็จไหม? ยังไม่ประสบความสำเร็จ แต่ว่าผมต้องการที่จะประสบความสำเร็จเพื่อที่จะอะไร? เพื่อที่จะรวย, รวยแล้วทำไม? รวยแล้วเพื่อจะได้สบาย แล้วใช้เวลาที่เหลือไปกับการใช้ชีวิตที่ไม่ต้องทำงานอยู่ในระบบเป็นกับดักหนู กับดักชีวิต ซึ่ง "It's not my path" (มันไม่ใช่ทางของผมนะ)

สิ่งที่ผมศึกษาแล้วก็ต่างออกไปเลยก็คือเรื่องการเงิน การเงิน การใช้ชีวิต การสร้างตัว ซึ่งธุรกิจเขาไม่ได้บอกว่าเออคุณต้องเป็นแบบนี้ เขาบอกประมาณว่าเออคุณอยากเป็นอะไรคุณก็เป็นไปเลย คุณอยากทำอะไรคุณก็ทำ ถามว่ามันต่างกันไหม แน่นอน ผมว่ามันต่างกันมากๆ แล้วมันดีด้วย

ถ้าคุณอยากจะเป็นในด้านนี้ โอเค คุณเป็นไป คุณอยากทำธุรกิจนี้ใช่ไหม โอเค ไอเดียนี้มันเวิร์คหรือเปล่า ก็ช่วยๆ กัน คุณทำได้ใช่ไหม อ่ะ คุณทำ คุณมีคำแนะนำอะไรไหม เขาก็จะแนะนำ เขาจะไม่มีการมาคอยกดให้อยู่ภายใต้การควบคุมของใครบางคนหรอก เขาจะมีแต่กันและกัน Support กัน ศึกษาด้วยกัน แลกก็ connection

นั่นแหละ สิ่งที่มันแตกต่างกันเลยระหว่างในโรงเรียนกับนอก

### Q8: เพื่อนรุ่นเดียวกัน — อะไรที่อยากเขย่าให้ตื่น

คือตอนนี้สิ่งที่ผมคิดเลยนะ ไอ้เหี้ย มึงอายุ 50 แล้วอ่ะ มึงควรจริงจังกับชีวิตได้แล้วนะเว้ย มึงไม่ควรที่จะเอ้อระเหยได้แล้วอ่ะ คือผมรู้สึกเลยนะ มันต่างกันมากๆ ตรงที่แนวคิดมันคนละแนวคิดกันเลย เขายังมัวแต่ติดเกมติดอนิเมะไม่จริงจังกับชีวิต ซึ่งมันขัดกับผมมากๆ ผมอ่ะไม่ได้เล่นบ่อย แต่นานๆ ทีเล่น คือไม่ได้เล่นจนติดขนาดนั้น คือเขามี mindset ที่ต่างกับผมมากจนผมรู้สึกว่าเออเขากับผมอ่ะคุยกันไม่ได้ คือเพื่อนแทบทั้งห้องอ่ะ ผมอ่ะมีเพื่อนอยู่กันไม่กี่คน นั่นแหละหรือคนเดียวด้วยซ้ำที่จริงอ่ะ

แล้วก็เรื่องการใช้ชีวิต เขาใช้ชีวิตแบบเออคุณอยากเล่นเกมใช่ไหม? เออกูก็เล่น กูอยากเล่น กูอยากเล่น ไม่มีการเอาจริงเอาจังกับชีวิตเลย

### Q9: ความเชื่อเรื่องเงินที่กูรูชอบขาย ที่คุณคิดว่าไม่จริง

ผมไม่รู้ ผมไม่รู้ ผมไม่รู้ ผมไม่รู้ ผมไม่รู้ ผมอยู่ในโลกธุรกิจมาสี่ปีก็จริง แต่ว่าผมไม่ได้เป็นคนที่เจาะลึกอะไรขนาดนั้น

คือด้วยความที่ผมเป็นเด็ก ผมก็ทำงานไปเรียนไปด้วย ผมไม่ได้ทำจนขนาดนั้น คือผมทำทำทำทำทำก็จริง แต่ว่าแบบไม่ได้เป็นคนที่โฟกัสเยอะขนาดนั้น ไม่ได้แบบลงลึกไปเลย เพราะว่าผมก็เรียนอยู่

ใช่ ถ้าธุรกิจมันง่าย ถ้าการหาเงินบนโลก Internet มันง่ายมาก ป่านนี้ก็ต้องมีคนทำไปเยอะแล้วก็จริง ใช่ป่ะล่ะ แต่ว่ามันไม่ได้ง่ายอย่างที่คิดเลย มันยาก มันยากมากๆ มันต้องใช้เวลา ขนาดผมยังต้องลองผิดลองถูกมาประมาณสามปี ถึงกว่าจะได้เรียนรู้ว่าเออ โอเคเราต้องทำแบบนี้นะ แล้วก็อยู่กับมัน ทำอย่างโน้นทำอย่างนี้ ซึ่งมันยาก แล้วก็ไม่มีใครแนะนำเลยว่าเออให้ออกมาทำธุรกิจกันนะ ไม่มีใครแนะนำเลย อย่างคุณ CK เจิง เขาก็บอกว่า "ผมไม่เคยแนะนำให้ใครทำธุรกิจเลย ธุรกิจมันเป็นเรื่องที่ยาก มันยากมากๆ" โอกาสเจ๊งมันสูง

ใช่ แล้วผมก็คิดแบบนั้นเหมือนกับเขาเลย

### Q10: คนเข้าใจ AI ผิดยังไงบ้าง

ผมคิดว่าสิ่งที่คนคาดหวังเนี่ย ส่วนใหญ่เขาคิดว่าแบบ เออ ใช้ AI แล้วรวย ใช้ AI แล้วแบบรวย AI คือทำให้คุณรวยได้เลยอะไรประมาณนั้น ถ้าใช้เป็นอะไรประมาณเนี้ย เขาคิดกันอย่างนั้น แต่ที่จริงผิดเลย

และก็ผมเห็นคนที่ใช้ AI แล้วมันยังผิดอยู่ก็คือเขาไปคาดหวัง เขาไปตั้งความหวังให้กับ AI มากเกินไปว่า เออ คุณน่ะช่วยให้ AI คิดสคริปต์ให้หน่อยแล้วก็ออกมาใช้ได้เลย หรือว่าให้มันแบบขุดไอเดีย ขุด ขัดเกลาสคริปต์อะไรประมาณนั้น

คือเขาไปหวังผลกับ prompt อันเดียวที่มันยาวๆ ที่มันแบบกำหนดไม่ชัดพูดซ้ำแต่เรื่องเดิมๆ ว่าแบบเออทำแบบนี้ๆ ใส่แบบนี้ๆ แต่ที่จริงแล้วอ่ะ ผมคิดว่าการที่อยากให้ AI ขัดเกลา script ให้มันดีอย่างแรก คุณต้องเริ่มจากการที่มี context ขอตัวเองก่อน

### Q17: บรรทัดแรก / ประโยคเปิด ของสคริปต์

# Hook ชอบกระแทก ขยี้ pain ตั้งคำถาม
เปิดแบบแรงๆ — กระแทกใส่หน้าคนดู
ชอบจิกคนอ่านก่อน "คุณคิดผิด" "คุณพลาด"
แล้วก็เอาตัวเองเป็นตัวอย่าง ยอมโชว์จุดอ่อน
ฮุกไม่ได้คิดจากศูนย์ — ดูคอนเทนต์คนอื่นมาเยอะจนจำ แล้วมาประยุกต์ใช้กับตัวเอง
มีไฟล์ฮุกเก็บไว้ด้วย

ตัวอย่าง:
- "ทำไม AI ตอบคุณได้โคตรกาก กับอีกคนที่งานออกมาดี ทั้งที่ไม่ได้พิมพ์คำสั่งเยอะเหมือนคุณ"
- "ผมเคยสมองฝ่อเพราะ AI ผมไม่ได้ขี้โม้ แต่ผมรู้สึกได้เลยว่าผมโง่ลงจริงๆ"
- "ถ้าคุณยังคิดว่า prompt ดี แล้ว output มันจะออกมาดี ผมบอกเลยครับคุณโคตรคิดผิด"
- "ถ้าคุณยังไม่เคยลองสิ่งนี้กับ AI ของคุณ ผมบอกเลยครับว่าคุณพลาดมาก"
- "ทำไมคอนเทคสั้นชัดถึงดีกว่า prompt ยาว"
- "คุณอาจจะโง่ลงโดยไม่รู้ตัวถ้าคุณยังใช้ AI แบบนี้"
- "สิ่งที่ผิดพลาดเกี่ยวกับผมที่มีต่อ AI เลยก็คือ…"
- "prompt ยาวก็ดีได้… จริงหรอ?"

**Hook Pattern:** [ถ้า/เลิก/ไม่มี] + [สิ่งที่คนทำผิด] + [ผลที่ตามมา]

### CTA ที่ใช้จริง
- CTA หลัก: "กดติดตามไว้แล้วเจอกันคลิปหน้า" (เปลี่ยนจากเดิม "DM คำว่า ai มาได้เลย")
- CTA รอง: "ถ้าคุณอยากเก่ง AI กว่าคน 90% กดติดตามไว้แล้วเจอกันคลิปหน้า"
- บางทีจบด้วยคำคมหรืออารมณ์แทน CTA
- ไม่ชอบ CTA ที่ขอทุกอย่างรวดเดียว "อย่าลืมกดไลค์กดแชร์กดติดตามกดเซฟ" — เยอะเกินสำหรับคลิปสั้น

### ภาษาและ Format
- ภาษาไทยเป็นหลัก คำศัพท์เทคนิคใช้อังกฤษ: prompt, hook, CTA, framework, content, AI agent, personal branding, digital product, pain point — แทบจะไม่ใช้คำเทคนิคเป็นภาษาไทยเลย "มันเอ๋อมาก"
- สคริปต์คลิป = เขียนยาวเป็นบทพูดเลย ไม่ใช่ bullet เพราะต้อง flow เหมือนพูดจริง ถ้า bullet มันติดขัด
- Caption IG = เขียนสั้นเป็นหัวข้อ เว้นวรรค ไม่ให้รกตา แทบไม่เขียน caption นอกจากโพสต์ภาพ
- คลิปยาว 60 วินาที ถึง 2 นาที
- ไม่ใช้ emoji (อาจจะใส่บ้างเล็กน้อยถ้าเป็นสไลด์)
- ไม่ใช้ hashtag

### Tone
- น้ำเสียงเดียวตลอด ไม่ว่าจะคุยกับใคร platform ไหน เป็นกันเอง อธิบายง่าย
- เพื่อนเล่าให้เพื่อนฟัง — ไม่ใช่ครูสอน ไม่ใช่กูรู
- ตรง ไม่อ้อมค้อม
- เวลาตื่นเต้น = พูดเร็วขึ้นนิดแต่โทนเดิม
- เวลา skeptical = จริงจังขึ้นหน่อย เปลี่ยนคำให้ตรงขึ้น มีเหตุผลรองรับเสมอ ไม่กลัวสวนกระแส
- ไม่เปลี่ยนน้ำเสียงตามสถานการณ์หรือตามคนที่คุย

### Transition words ที่ใช้บ่อย
ซึ่ง, แต่ว่า, ดังนั้น, และ, so, นั่นแหละ, แบบ, ประมาณ

### คำติดปาก
"ถ้าให้พูดง่ายๆเลยก็คือ", "ขอพูดตามตรงนะ", "เอาจริงๆนะ", "คุณพอเข้าใจใช่ไหมครับ", "อ่าใช่", "แบบ", "so", "จริงๆแล้ว", "ง่ายๆเลยก็คือ", "พูดตรงๆเลย", "นั่นแหละครับ", "ประมาณนั้นแหละ", "แค่นั้นเอง"

**หมายเหตุ:** "ถ้าให้พูดง่ายๆเลยก็คือ" ใช้เพื่อสรุปดักคนที่อาจจะหลุดหรือลืม ถ้าเขาเข้าใจแล้วจะไม่พูดซ้ำ

### การ "ครับ"
ใช้ "ครับ" บ้าง แต่ไม่เยอะเกินไป เยอะเกินก็ไม่ดี น้อยเกินก็ไม่ดี

### เรียกคนดู
ใช้ "คุณ" — ครอบคลุมทั้งผู้หญิงและผู้ชาย

### คำหยาบ
ใช้ได้แต่เลือกใช้ — "แม่ง" ใช้ได้เพื่อเพิ่มน้ำหนักให้กับความเร่งด่วน เช่น "ถ้าคุณไม่ใช้ Claude ในตอนนี้ คุณแม่งกำลังจะพลาดโอกาส"
เส้นแบ่ง: "มึง กู ไอ้เหี้ย" ไม่ใช้เด็ดขาด — เพราะกลุ่มเป้าหมายอายุมากกว่า ยังต้องให้เกียรติ

### วิธีตัดงาน — "การเขียนที่ดีที่สุดคือการลบ"
- ตัดคำขยายที่ซ้ำ — อธิบายไปแล้วรอบนึงก็พอ ไม่ต้องวนซ้ำ
- ตัดประโยคที่ซ้ำความหมาย
- ตัดตัวอย่างให้เหลือแค่อันเดียว
- ตัดคำที่ไม่จำเป็น
- สายกลาง — เนื้อเยอะแต่น้ำน้อย
- ตัวกรอง: "ถ้าเราเป็นคนอ่าน อยากอ่านเหรอวะมันยาวขนาดนี้"

### เวลาเขียนติด

คืออย่างแรก แม่งเล่ายาวชิบหาย Intro พ่อมึงโคตรยาวแล้วก็เนื้อหา เนื้อหาแบบปกติ เนื้อหามันจะมีแค่ประมาณแบบถ้าเอาแบบ 10% เนื้อหา 10% แต่แม่งบวกไปอีกประมาณ 50% อย่างเงี๊ยะ 30-20 ประมาณน่ะ เออ ประมาณเท่าตัวยังพอทน แต่อันเนี้ยมันแบบรากยาว คือหลงไปในน้ำอย่างเดียวอะ เป็น content ที่มีน้ำแต่ไม่มีเนื้อ หรือว่าเนื้อน้อย เอ้ย เออ เนื้อน้อย

คือ ทำทรงแบบโอ้โห ดูคลิปฉลาดมาก แต่พูดยาวไปเรื่อย คือ แม่งนั่งหงุดหงิดชิบหาย

### Q47: ขโมยความรู้

เอาจริงๆ ผมก็รู้สึกโกรธ แต่ก็รู้สึกเฉยๆ เพราะว่า มึงเอาความรู้ของกูไป แต่มึงถ้าไม่เปลี่ยนคำอย่างเงียบ ผมก็โกรธอยู่ แต่ถ้ามันเปลี่ยนคำพูด, เปลี่ยนอะไรต่างๆ อย่างเงียบ, เปลี่ยนการนำเสนอ, เปลี่ยนการพูดอย่างเงียบ ผมก็ยินดีอยู่ ก็ไม่ได้ว่าอะไร

### Q48: Hook ที่แย่ที่สุด

สวัสดีครับ ผมชื่อ และ Hook ที่มันไม่ค่อยน่าฟังเท่าไหร่ก็คือ Hook ที่มันแบบ over หรือว่า Hook ที่ไม่ดึงดูด สองอย่างนี้มันจะต่างกันมาก:

1. Hook ที่มัน overrate มันจะแบบ over จน โอ้ แม่งไม่น่าฟังเลยอ่ะ
2. Hook ที่ไม่ดึงดูด คือ Hook ที่มันแบบธรรมดาเฉยๆ หรือว่าแบบคำพูดที่มันแบบ เออ ไม่กระแทก หรือว่าไม่ดึงดูดคนดู

### Q51: เสียงตอนตื่นเต้น

เอาจริงๆ นะ ตอนผมได้เงินจริงๆ ผมดีใจมากๆ ตื่นเต้นสุดๆ แบบ fucking shit มันเป็นแบบคราวแรกที่ผมทำเงินได้หรือว่าแบบตั้งใจทำเงินใช่ไหม? มันตื่นเต้นมากๆ หรือว่าได้เงินอย่างเนี๊ยะ ผมจะแบบพูดเร็วขึ้นหรือว่าแบบตื่นเต้นมากๆ ที่แบบ ไอ้เชี่ยกูทำได้แล้วๆๆ อะไรต่างๆ ความเร็วเปลี่ยน แล้วก็อาจจะมีคำหยาบมากขึ้นถ้าผมทำดีลได้สำเร็จอ่ะ คือแบบมันตื่นเต้นจริงๆ นะ เว้ย ตอนได้เงินอ่ะ แม่งเป็นโมเมนต์ที่รู้สึกว่า เชี่ยกูทำมาตั้งนานกูได้แล้ว อะไรประมาณนั้น

### Q52: เสียงตอน skeptical

ภายนอกหรอ ผมก็จะพูดด้วยตามสิ่งว่า เอิ่มก็พูดปกติน่ะแหละ แต่ก็พูดให้มันช้าลงหรือว่าพยายามเจาะลึกเข้าไปอีก เพราะที่ๆ นั้น พยายามตอบให้ลึกมากขึ้น ถ้าคุณตอบลึกอีกไม่ได้ ก็แสดงว่า คุณอาจจะมีแบบ เขา คุณอาจจะแบบพูดมั่วหรือเปล่า หรือว่าคุณไม่มีความรู้จริงหรือเปล่าอะไรประมาณนั้น

ส่วนภายในใจผมก็คิดว่า ถ้าคนนี้ไม่รู้จริงๆ เราลองพยายามตอบให้ถามคำถามลึกลงไปอีก เพื่อที่จะให้เขาตอบ แต่ถ้าเขาตอบได้ก็ถือว่าเขาแม่งถูกต้อง

### Q53: เวลาคนไม่เห็นด้วย

ก็ถ้าเป็นคอมเมนต์แย่ๆ ผมก็อาจจะไม่ตอบ

แล้วถ้าผมผิดจริงๆ ผมก็จะยอมรับว่าผิด เออ ผมคิดอย่างนี้ ผมผิด

แต่ถ้าเขาเห็นต่างผม ก็จะลองแบบเออมาปรับความคิดดูว่า เออ ทำไมเขาคิดอย่างนั้น ก็ลองมา inside ดูว่าผมกับเขาต่างกันยังไง แล้วก็สามารถประยุกต์อะไรได้บ้าง

### Q54: หงุดหงิด/โมโหในงานเขียน

### Q55: คุยกับพ่อแม่ / เพื่อน / audience

ก็เพื่อนน่ะ ผมใช้คุณกับผม พ่อแม่ก็ภาษาเดียวกัน ก็พูดโทนเดียวกัน ก็แบบโทนง่ายๆ เข้าใจง่ายๆ หรือว่าแบบพวกไม่ได้มีแบบคำ Over อะไรต่างๆ แค่ให้มันแบบเอาฟังดูเข้าใจ

[ใช้ "คุณ/ผม" กับทุกคน รวมเพื่อน — ไม่ใช้ กู/มึง ในชีวิตปกติ ยกเว้นเวลาตื่นเต้นพูดในใจ]

### Q56: DNA ของเสียงคุณ

ความเป็นเพื่อน ความสบายๆ นั่นแหละคือความเป็นตัวผม แล้วก็สิ่งที่เป็นตัวผมอีกก็คือคำที่ผมชอบใช้บ่อยมากที่สุด แล้วก็การพูด

### Q57: โมเมนต์ที่สวมบทบาท

มีดิๆๆๆๆ ก็คือการที่ผมทำคลิปช่องนึง, อะคือช่องเก่าของผม, มันเป็นช่องเก่าของผม, ผู้ชมประมาณ 1 แสน 2 หมื่นคน แต่ช่องนั้นเลิกทำไป, มันเป็นการให้ความรู้แล้วก็ให้พัฒนาตัวเองและก็ให้แรงบันดาลใจ เพราะตอนนั้นผมชอบออกกำลังกายมากๆ ผมก็เลยทำแม่งเลย

มันขัดกับผมตัวจริงมากๆ ถ้าเปรียบเทียบในตอนนี้กับแต่ก่อน ถ้าแต่ก่อนน่ะ มันก็เออพอเข้าใจง่าย เอ้ย พอเข้าใจได้อยู่ เพราะว่าแต่ก่อนผมก็ชอบออกกำลังกายแล้วกับนิสัยมันก็ต่างกับตอนนี้ นิสัยตอนนั้นผมพูดหยาบแล้วก็ไม่ค่อยดี นิสัยดีเท่าไหร่

### Q58: อะไรเปลี่ยนคุณ

ก็มันจะเป็นเรื่องที่ผมเก็บกดอะนะ ก็คือเรื่องเก่าๆต่างๆที่ทำร้ายจิตใจของผมเนี๊ยะ จนผมไม่ได้รู้สึกแล้ว ก็คือไม่ได้รู้สึกกลับไปเป็นคนอ่อนแอเหมือนตอนนาน ก็คือโดนความคาดว่าไม่แคร์อะไรประมาณนาน เออ

แล้วก็มีเหตุการณ์หนึ่งก็คือ แต่ก่อนผมเป็นคนที่ติดเกมมากๆ แต่ว่าหลังเริ่มพัฒนาตัวเอง ชีวิตผมก็เปลี่ยนไป ก็คือผมเป็นคนที่ดีขึ้น จิตใจแข็งแกร่งขึ้น และนิสัยเปลี่ยนไปมากขึ้น จากตอนนั้นที่เป็นคนที่ไม่เอาไหนเลย ก็เริ่มทำงาน เริ่มทำนู่น ทำนี่ แข็งแรง

### Q59: มอง audience ยังไง

Audience ของผมจะเป็นคนที่อายุมากกว่าผมก็จริง แต่ก็คือเป็นการคุย ผมมองพวกเขาเป็นเพื่อน คือมองพวกเขาแบบเอาเป็นเพื่อนกันนะ แบ่งปันความรู้, แบ่งปันสิ่งที่เข้าใจแล้วก็แชร์สิ่งที่ผมดูให้กับเขาแบบฟรีๆ ง่ายๆ ส่วนเรื่องเงินค่อยว่ากันอีกทีอะไรประมาณนั้น

และผมอยากจะบอกว่าสิ่งที่เราคุยกันมาทั้งหมดเนี่ย ผมใช้พูดหมดเลย ไม่มีพิมพ์

### Q60: อะไรทำให้แตกต่างจากเด็ก 16 คนอื่น

ผมไม่รู้อ่ะ ผมไม่ได้ขยัน แต่ว่าผมพยายามที่จะเป็นแบบคนที่พยายามวางแผนแล้วก็จัดระเบียบให้มันเข้าใจง่าย ก็คือผมเป็นคนที่พยายามวางแผนให้รอบคอบ, ให้เรียบร้อยแล้วก็ให้มีระบบที่เรียบง่าย แล้วก็เริ่มทำจริงๆ

ในขณะที่คนอื่นทำทำทำทำทำสม่ำเสมอ ทำไม่มีระบบที่เป็นเหมือนผม ก็ผมว่ามันแล้วแต่คนอ่ะ แต่ผมคิดว่าเป็นข้อต่างนะ เพราะว่าผมเป็นคนที่ตั้งแผนเรียนรู้งานเยอะมากๆ แล้วก็พยายามทำให้มันง่าย เอาให้ง่าย คือ ทำยังไงก็ได้มันง่ายมากที่สุด โดยที่ผมอ่ะ ไม่ต้องทำอะไรเยอะมากๆ

### Q61: เปรียบกับตัวละครอะไร

เป็นเพื่อนสนิทที่อธิบายเรื่องที่เข้าใจยาก ให้มันเข้าใจง่ายเหมือนสอนการบ้านที่เข้าใจยากๆ ให้เข้าใจง่าย และยกตัวอย่าง

## QUICK REFERENCE CARD

### Always:
- เปิดด้วย hook กระแทก ขยี้ pain ตั้งคำถาม หรือใช้ตัวเองเป็นเหยื่อ
- ใช้ Pattern: [ถ้า/เลิก/ไม่มี] + [สิ่งที่คนทำผิด] + [ผลที่ตามมา]
- ใส่ประสบการณ์ตัวเองในทุก piece — "ผมเคย..." เป็น move หลัก
- เริ่มจาก context ของตัวเองก่อนเสมอ (About Me)
- โทนเดียว: เพื่อนเล่าให้เพื่อนฟัง — peer-to-peer, ไม่ใช่ครู ไม่ใช่กูรู
- ใช้ "คุณ" เรียก audience ทุกครั้ง
- ใช้ "ครับ" บ้างพอเหมาะ ไม่เยอะ ไม่น้อย
- จังหวะประโยคผสม สั้น ยาว สลับ
- ภาษาอังกฤษได้ แต่ห้ามมากกว่าไทย
- คำเทคนิคใช้อังกฤษเสมอ: prompt, hook, CTA, framework, content, AI agent, personal branding, digital product, pain point
- เปลี่ยนความเร็วในการพูด (เร็ว/ช้า)
- ตัดสิ่งที่ซ้ำ ตัดน้ำ เหลือเนื้อ
- ใช้ filter "ถ้าเป็นคนอ่าน อยากอ่านมั้ย"
- ทำ outline ก่อนเขียน (โครง / Framework / Hook / Body / CTA)
- ปล่อย AI draft, แต่รีวิวเองรอบสุดท้าย: "นี่เป็นเสียงผมไหม?"
- คลิป 60-120 วินาที, สัดส่วน Hook 10% / เนื้อ 70% / ปิด 20%

### Never:
- ห้าม "สวัสดีครับ ผมชื่อ..." / "วันนี้อยู่กับช่อง..." / "ขอขอบคุณที่ดูจนจบ"
- ห้ามคำกูรู ("ถ้าคุณไม่มีวันยอมแพ้ ยังไงคุณก็ชนะ")
- ห้ามคำติด trend TikTok ("ปังปุริเย่", "บ่งบอก")
- ห้ามใช้ "มึง กู ไอ้เหี้ย" ในงาน (ใช้ "แม่ง" ได้พอประมาณ)
- ห้ามภาษาอังกฤษเยอะกว่าไทย
- ห้าม emoji (ยกเว้นสไลด์เล็กน้อย)
- ห้าม hashtag
- ห้ามเครื่องหมายวรรคตอน
- ห้ามอารมณ์ขัน
- ห้ามจังหวะสม่ำเสมอเกินไป (= AI feel)
- ห้ามใช้คำที่ "คนปกติไม่ใช้" (AI tell)
- ห้ามใช้ AI สร้างรูปคุณ
- ห้ามแตะ: การเมือง, การพนัน, นอกความรู้ AI
- ห้ามเรียกชื่อครีเอเตอร์อื่นในคลิป
- ห้ามเปรียบเทียบ/ด่าใครเปิดเผย
- ห้ามใช้ analogy: เพศ, ความรุนแรง, ศาสนา
- ห้ามดูถูก audience
- ห้าม shame clip
- ห้าม fake scarcity / rags-to-riches / Lambo flex / testimonial ปลอม
- ห้าม "ราคา 50K → 999" เว้นแต่ overprice จริง
- ห้าม CTA แบบ "ไลค์ แชร์ ติดตาม เซฟ" รวดเดียว
