# Content System — AI Runtime Version

Use the right framework for the right format.

## Format Map
- Post / Article → Content Framework
- Personal Story → Story Framework
- Thread → Thread Framework
- Carousel → Carousel Framework
- Short Video → Video Framework
- One idea → many pieces → Content Multiplier

---

## Universal Rules
- one idea per piece
- short sentences
- clarity beats complexity
- no hype or empty marketing language
- reader should understand the idea in one read
- challenge beliefs when useful
- use concrete examples
- end with a memorable takeaway

---

## 1) Content Framework
**Use for:** standard posts and articles
**Flow:** Hook → Context → Breakdown → Example → Insight

Rules:
- Hook should challenge a belief, expose a mistake, or reveal a hidden truth
- Context should explain why this topic matters and what confusion exists
- Breakdown should simplify the idea one part at a time
- Example should make the idea concrete
- Insight should end with the core takeaway

---

## 2) Story Framework
**Use for:** personal experience posts
**Flow:** Hook → Situation → Problem → Turning Point → Lesson → Insight

Rules:
- start with a mistake, realization, or surprising experience
- make the situation clear quickly
- show what went wrong and why it mattered
- turning point = the moment perspective changed
- story exists to teach the reader something, not to impress
- end with a simple lesson people can remember

---

## 3) Thread Framework
**Use for:** X / Twitter threads
**Flow:** Hook → Context → Belief → Breakdown → Examples → Insight → Conclusion

Rules:
- thread should feel like a thinking journey, not random tips
- tweet 1 must challenge a belief clearly
- context explains why the topic matters now
- state the common belief before breaking it
- each tweet in breakdown should carry one insight
- examples make the idea real
- final insight should be memorable
- tone should feel like conversation, not lecture

---

## 4) Carousel Framework
**Use for:** slide-based visual content
**Flow:** Hook → Problem → Reframe → Breakdown → Example → Insight → Takeaway

Rules:
- each slide = one idea
- slide 1 must stop the scroll
- slide 2 shows what people do wrong
- slide 3 introduces the new perspective
- middle slides simplify the idea step by step
- final slides deliver the realization and takeaway
- carousel should guide reader from confusion to clarity

---

## 5) Video Framework
**Use for:** short-form video scripts (30–60 sec)
**Flow:** Hook → Problem → Explanation → Example → Takeaway

Rules:
- Hook: 0–3 sec, strong opening statement
- Problem: show what people are doing wrong
- Explanation: teach the real idea simply
- Example: use one simple scenario
- Takeaway: one memorable closing line

---

## 6) Content Multiplier
**Use for:** turning one idea into many pieces
**Principle:** don't chase more ideas, extract more value from each one

Process:
1. Start with one clear belief-challenging idea
2. Ask: what belief does this challenge, why do people believe it, what is the better view, what example proves it, what should people remember?
3. Create multiple hooks from the same idea
4. Expand into multiple formats
5. Swap examples to create more variations

---

## Final Writing Standard
Every piece should:
- sound human
- be easy to follow
- teach one real thing
- avoid fluff
- leave the reader with a sharper thought than before
