# 🎬 Subtitle Studio

เว็บแอปถอดเสียงพูด **ภาษาไทย** ออกจากวิดีโอ แล้วทำเป็นซับไตเติลในหน้าตัดต่อแบบ
timeline เหมาะกับการทำซับสไตล์ **Reels / Shorts / TikTok** (ขึ้นทีละ 2–3 คำ เปลี่ยนไว ๆ)

- ดึง **เฉพาะเสียง** ออกจากคลิป → ไฟล์ใหญ่กว่า 100 MB ก็ทำงานในเครื่องได้ (ไม่อัปโหลดวิดีโอขึ้นเซิร์ฟเวอร์)
- ถอดเสียงด้วย **Google Gemini** — ถอดเฉพาะ "คำพูดของมนุษย์ภาษาไทย" ไม่เอาเสียงสัตว์ / ดนตรี / สภาพแวดล้อม
- หน้าตัดต่อแบบ timeline: ลากย้าย / ยืด-หด / เลือกหลายท่อน / ตัด / รวม ได้เหมือนโปรแกรมตัดต่อ
- ฟอนต์ **[K2D](https://fonts.google.com/specimen/K2D)** ปรับขนาด สี ขอบ เงา ตำแหน่ง ได้
- **อนิเมชันคำขึ้น** สไตล์ Reels: เด้ง (pop) / เลื่อนขึ้น / ขึ้นทีละคำ (แยกคำไทยด้วย `Intl.Segmenter`)
- ใช้ได้บน **iPad / แท็บเล็ต** (ลากด้วยนิ้วได้, ปุ่มทัชใหญ่ขึ้น) — แนะนำแนวนอน
- ทุกอย่างรันใน browser (client-side ล้วน) — API key เก็บใน `localStorage` ของเครื่องคุณเอง เปลี่ยนได้ทุกเมื่อ

## เริ่มใช้งาน

ต้องมี **Node.js 18+** (แนะนำ 20/22)

```bash
cd subtitle-studio
npm install
npm run dev
```

เปิด `http://localhost:5173`

สร้างไฟล์สำหรับ deploy:

```bash
npm run build      # ได้ผลลัพธ์ในโฟลเดอร์ dist/ (เป็น static site เอาไปวางที่ไหนก็ได้)
npm run preview    # ลองเปิด build ที่ http://localhost:4173
```

> เป็น static site ล้วน เอาไป deploy บน Vercel / Netlify / GitHub Pages / Cloudflare Pages ได้เลย

2. กด *Create API key* แล้วคัดลอกค่า (ขึ้นต้นด้วย `AIza…`)
3. ในแอป กดปุ่ม **⚙️ ตั้งค่า** มุมขวาบน → วางคีย์ → บันทึก

คีย์ถูกเก็บไว้ใน `localStorage` ของเบราว์เซอร์เครื่องนี้เท่านั้น ไม่ถูกส่งไปไหนนอกจาก Google

## วิธีใช้

1. **ลากวิดีโอมาวาง** (หรือคลิกเลือกไฟล์) — รองรับ MP4 / MOV / MKV / WEBM / ไฟล์เสียง
2. ตั้ง **จำนวนคำต่อท่อน** (Reels แนะนำ `2`) แล้วกด **ดึงเสียง + ถอดเสียง**
   - ระบบดึงเสียง (mono 16 kHz) ด้วย FFmpeg ในเครื่อง → ส่งเฉพาะเสียงให้ Gemini ถอดความ
3. ได้ซับเป็นท่อน ๆ บน timeline พร้อม **คลื่นเสียง** ช่วยเล็งจังหวะ
4. **ตัดต่อซับ** ได้เต็มที่:
   | ทำอะไร | วิธี |
   |---|---|
   | ย้ายตำแหน่งซับ (เวลา) | ลากตัวบล็อกบน timeline |
   | ยืด/หดความยาว | ลากขอบซ้าย-ขวาของบล็อก |
   | เลือกหลายท่อน | กด **Shift/⌘** คลิก หรือ **ลากคลุม** พื้นที่ว่าง |
   | ย้ายหลายท่อนพร้อมกัน | เลือกหลายท่อนแล้วลาก |
   | แก้ข้อความเร็ว | **ดับเบิลคลิก** ที่ท่อน หรือแก้ในแผง "แก้ไขซับ" |
   | ตัดท่อน | วางหัวเล่นตรงจุดที่ต้องการ → กด **✂️ ตัด** |
   | รวมท่อน | เลือกหลายท่อน → **🔗 รวมท่อน** |
   | ลบ | เลือกแล้วกด **🗑️** หรือปุ่ม `Delete` |
5. ปรับ **สไตล์ซับ (K2D)** — น้ำหนักฟอนต์ ขนาด สี ขอบ เงา ตำแหน่ง (ลากซับบนวิดีโอเพื่อวางเองก็ได้)
   และเลือก **อนิเมชันคำขึ้น**: เด้ง / เลื่อนขึ้น / ขึ้นทีละคำ + ปรับความเร็วได้
6. **ส่งออก** เป็น `.srt` / `.vtt` หรือคัดลอก JSON ไปใช้ต่อ
7. อยากเริ่มคลิปใหม่ → กด **🗑️ ล้างคลิป** มุมขวาบน

### ใช้บน iPad / แท็บเล็ต
- ใช้ได้บน Safari (iPadOS 16+) — แนะนำถือ **แนวนอน** จะกว้างพอ
- แตะที่ท่อนซับเพื่อเลือก → แก้ข้อความในแผง “แก้ไขซับ” · ลากบล็อกเพื่อย้าย/ยืดได้ด้วยนิ้ว
- ไฟล์ที่ใหญ่มาก (หลายร้อย MB) อาจกินหน่วยความจำเกินขีดจำกัดของ Safari บน iOS ได้ —
  ถ้าดึงเสียงไม่สำเร็จให้ลองตัดคลิปให้สั้นลงก่อน (บนเครื่องคอมจะรับไฟล์ใหญ่ได้ดีกว่า)

### เปลี่ยน API key
กด **⚙️ ตั้งค่า** เพื่อแก้/แทนที่คีย์ได้ทุกเมื่อ (เช่น คีย์หมดอายุ) หรือกด **ลบคีย์** เพื่อล้างออก

### คีย์ลัด
- `Space` — เล่น/หยุด
- `Delete` / `Backspace` — ลบท่อนที่เลือก

## พรอมต์ที่ใช้ถอดเสียง

ส่งให้ Gemini พร้อมไฟล์เสียง (ค่า `${wordsPerChunk}` มาจากช่อง "จำนวนคำต่อท่อน"):

> ถอดเสียงเฉพาะ 'คำพูดของมนุษย์' ที่เป็นภาษาไทยในคลิปนี้เท่านั้น ห้ามถอดเสียงสัตว์ (เช่น เสียงแมว หมา),
> เสียงดนตรี, หรือเสียงสภาพแวดล้อมอย่างเด็ดขาด แบ่งคำพูดเป็นท่อนสั้นๆ ตามจังหวะการหายใจหรือการเว้นวรรค
> ตามธรรมชาติ โดยให้แต่ละท่อนมีความยาวประมาณ `${wordsPerChunk}` คำ … พร้อมระบุเวลาเริ่ม (start) และจบ (end)
> เป็นวินาที … หากช่วงไหนไม่มีเสียงคนพูด ห้ามสร้างข้อมูลของช่วงนั้นมาเด็ดขาด

โมเดลถูกบังคับให้ตอบเป็น JSON (`responseSchema`) → `[{ text, start, end }]` แล้วแอปจะกรอง/จัดเรียงให้อัตโนมัติ

## โครงสร้าง

```
src/
  App.tsx              จัดเลย์เอาต์ + ไปป์ไลน์ (ดึงเสียง → ถอด → ใส่ timeline) + คีย์ลัด
  store.ts             สถานะทั้งหมด (Zustand) — คลิป/เลือก/เล่น/สไตล์/ตั้งค่า
  lib/
    ffmpeg.ts          ดึงเสียงเป็น mp3 mono 16k ด้วย FFmpeg.wasm
    gemini.ts          เรียก Gemini + พรอมต์ + แปลง/กรองผลเป็น cues
    waveform.ts        ถอดคลื่นเสียงไว้วาดบน timeline
    export.ts          ส่งออก SRT / VTT / JSON
    time.ts            จัดรูปแบบเวลา
  components/
    Uploader · VideoStage · Timeline · ControlBar
    TranscribePanel · Inspector · StylePanel · SettingsModal
```

## หมายเหตุทางเทคนิค

- ใช้ FFmpeg core แบบ **single-thread** (โหลดจาก unpkg) จึงไม่ต้องตั้ง header COOP/COEP
  — ทำให้โหลด Google Fonts (K2D) ได้ตามปกติ
- การส่งเสียงให้ Gemini เป็นแบบ **inline** เหมาะกับคลิปสั้น (~ไม่เกิน 25–30 นาที ที่ mono 16k)
  ถ้ายาวกว่านั้นให้ตัดคลิปก่อน
- ซับจะ scale ตามขนาดวิดีโอด้วยหน่วย container-query (`cqh`) — preview ตรงกับสัดส่วนจริงเสมอ

### อยากต่อยอด
- เผาซับลงวิดีโอ (burn-in) เป็น MP4 ด้วย FFmpeg.wasm
- รองรับคลิปยาวด้วย Gemini File API (resumable upload)
- บันทึก/เปิดโปรเจกต์ (.json)
